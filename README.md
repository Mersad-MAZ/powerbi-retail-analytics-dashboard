# Power BI Retail Analytics Dashboard

## Overview

This project showcases an interactive Power BI dashboard built using the Maven Market dataset. The dashboard provides insights into retail performance through sales analysis, customer behavior, store performance monitoring, and return tracking.

The project demonstrates data modeling, DAX calculations, KPI development, and interactive dashboard design using Power BI.

---

## Project Objectives

* Monitor retail sales performance.
* Evaluate store and regional performance.
* Analyze customer purchasing behavior.
* Track product sales and returns.
* Develop business KPIs for decision-making.

---

## Dataset

The dashboard is built using retail business data, including:

* Transactions
* Customers
* Products
* Stores
* Regions
* Returns
* Calendar
